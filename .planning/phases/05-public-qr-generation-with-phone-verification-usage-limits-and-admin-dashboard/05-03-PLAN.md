1. Create `src/app/actions/create-public-qr.ts`:
   - `'use server'`
   - Import `createAdminClient` from `@/lib/supabase/admin`
   - Import `cookies` from `next/headers`
   - Import `generateQrDataUrl` from `@/lib/qr-generator`
   - Define `FREE_LIMIT = 5`
   - Define slug generator: `function generatePublicSlug(phone: string): string` — format `{last4}-{random6}` where random6 = `Math.random().toString(36).slice(2, 8)`. Example: "4567-k3r9xm"
   - Export type `CreatePublicQrResult = { error?: string; gate?: boolean; qrData?: { slug: string; dataUrl: string; label: string } }`
   - Export `async function createPublicQr(message: string | null): Promise<CreatePublicQrResult>`:
     a. Read verified_phone cookie — if missing, return `{ error: 'Phone not verified' }`
     b. Create admin client
     c. Check phone_usage: `admin.from('phone_usage').select('usage_count').eq('phone_number', phone).single()`
     d. If `usage_count >= FREE_LIMIT`: return `{ gate: true, error: 'You have used your 5 free QR codes. Sign up with Google to continue.' }`
     e. Generate slug via `generatePublicSlug(phone)` — check uniqueness with a select; if taken, regenerate (retry up to 3 times)
     f. Insert into qr_codes using admin client (bypasses RLS since user_id is null):
        ```
        { user_id: null, phone_number: phone, slug, label: `QR ${(usage?.usage_count ?? 0) + 1}`,
          platform: 'whatsapp', contact_target: phone, default_message: message,
          is_active: true, scan_count: 0 }
        ```
     g. Upsert phone_usage: increment usage_count
        ```
        admin.from('phone_usage').upsert(
          { phone_number: phone, usage_count: (usage?.usage_count ?? 0) + 1, updated_at: new Date().toISOString() },
          { onConflict: 'phone_number' }
        )
        ```
     h. Generate QR data URL: `await generateQrDataUrl(slug)`
     i. Return `{ qrData: { slug, dataUrl, label } }`
