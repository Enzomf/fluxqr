  <action>
Create `src/app/dashboard/new/actions.ts` with `'use server'` directive.

Define a `FormState` type:
```typescript
type FormState = {
  errors?: { [field: string]: string[] }
  message?: string | null
}
```

Define the Zod schema `CreateQrSchema`:
- `label`: `z.string().min(1, 'Label is required').max(100, 'Label too long')`
- `slug`: `z.string().min(1, 'Slug is required').regex(/^[a-z0-9-]+$/, 'Only lowercase letters, numbers, and hyphens')` with `.min(2, 'At least 2 characters').max(50, 'Max 50 characters')`
- `platform`: `z.enum(['whatsapp', 'sms', 'telegram'], { message: 'Select a platform' })`
- `contact_target`: `z.string().min(1, 'Contact target is required')`
- `default_message`: `z.string().optional().default('')`

Export `createQrCode(prevState: FormState, formData: FormData): Promise<FormState>`:
1. Parse form data with `CreateQrSchema.safeParse(Object.fromEntries(formData))`.
2. If invalid, return `{ errors: validated.error.flatten().fieldErrors }`.
3. Call `createClient()`, then `supabase.auth.getUser()`. If no user, `redirect('/login')`.
4. Insert into `qr_codes` with `{ ...validated.data, user_id: user.id }`.
5. If error code is `'23505'`, return `{ errors: { slug: ['This slug is already taken'] } }`.
6. If any other error, return `{ message: 'Failed to create QR code. Please try again.' }`.
7. Call `revalidatePath('/dashboard')`.
8. Call `redirect('/dashboard')` — MUST be outside any try/catch block (Next.js redirect throws internally).

Import `z` from `'zod'`, `redirect` from `'next/navigation'`, `revalidatePath` from `'next/cache'`, `createClient` from `'@/lib/supabase/server'`.
  </action>

  </verify>
