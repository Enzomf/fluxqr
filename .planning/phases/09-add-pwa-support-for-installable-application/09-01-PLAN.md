<task type="auto">
  <name>Task 2: Configure Serwist in next.config.ts, update tsconfig.json, .gitignore, and add Apple meta to layout.tsx</name>
  <files>next.config.ts, tsconfig.json, .gitignore, package.json, src/app/layout.tsx</files>
  <action>
1. Update `next.config.ts` — wrap with Serwist:
   ```typescript
   import { spawnSync } from "node:child_process";
   import withSerwistInit from "@serwist/next";
   import type { NextConfig } from "next";

   const revision =
     spawnSync("git", ["rev-parse", "HEAD"], { encoding: "utf-8" }).stdout.trim() ||
     crypto.randomUUID();

   const withSerwist = withSerwistInit({
     swSrc: "src/app/sw.ts",
     swDest: "public/sw.js",
     disable: process.env.NODE_ENV === "development",
     reloadOnOnline: false,
     additionalPrecacheEntries: [{ url: "/~offline", revision }],
   });

   const nextConfig: NextConfig = {};

   export default withSerwist(nextConfig);
   ```
   Note: `reloadOnOnline: false` per research — avoids destroying form state when user comes back online. `additionalPrecacheEntries` ensures offline page is precached before user goes offline.

2. Update `tsconfig.json`:
   - Add `"webworker"` to `compilerOptions.lib` array: `["dom", "dom.iterable", "esnext", "webworker"]`
   - Add `"@serwist/next/typings"` to `compilerOptions.types` array: `["vitest/globals", "@serwist/next/typings"]`
   - Add `"public/sw.js"` to `exclude` array: `["node_modules", "public/sw.js"]`

3. Update `package.json` scripts:
   - Change `"build"` from `"next build"` to `"next build --webpack"` (required: Serwist webpack plugin does NOT run under Turbopack which is the Next.js 16 default)

4. Update `.gitignore` — add at the end:
   ```
   # PWA generated files
   public/sw.js
   public/sw.js.map
   public/swe-worker*
   ```

5. Update `src/app/layout.tsx` — add Apple PWA meta tags to the existing `metadata` export:
   - Add `icons.apple` array with `{ url: "/icon-192x192.png", sizes: "192x192", type: "image/png" }`
   - Add `appleWebApp` object with `{ capable: true, title: "FluxQR", statusBarStyle: "black-translucent" }`
   - Do NOT modify the existing metadataBase, title, description, openGraph, or twitter fields
   - Do NOT modify the RootLayout component body — only the metadata export changes
  </action>

  </verify>
